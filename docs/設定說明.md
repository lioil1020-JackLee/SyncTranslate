# SyncTranslate v2 設定說明

設定來源為 `config.yaml`；範例在 `config.example.yaml`。讀取流程會經過 schema parser、migration 與 dataclass schema。

## Product Mode

```yaml
runtime:
  session_mode: meeting   # meeting | dialogue
```

- `meeting`: 純監聽字幕、翻譯、記錄模式，不需要 virtual driver 或 bridge。
- `dialogue`: 雙向語音翻譯模式，需要 SyncTranslate Virtual Speaker / Microphone 與 bridge。

舊版 `direction.mode` 只保留 compatibility，不再是 v2 routing policy 的核心語意。

## Meeting Config

```yaml
meeting:
  audio_source: system_input          # system_input | system_output_loopback
  input_device: ""
  output_loopback_device: ""
  asr_language: zh-TW                 # zh-TW | en | ja | ko | th
  translation_target: en              # zh-TW | en | ja | ko | th
  record_transcript: true
  tts_enabled: false
```

meeting mode 中 `tts_enabled: false` 不會關閉 ASR/translation/transcript。它只代表不輸出 TTS 音訊。

## Dialogue Config

```yaml
dialogue:
  remote_asr_language: en
  local_asr_language: zh-TW
  remote_translation_target: zh-TW
  local_translation_target: en
  remote_tts_voice: none
  local_tts_voice: none
  passthrough_mode: direct
  passthrough_gain: 1.0
```

dialogue mode 中某方向 `tts_voice: none` 代表該方向走 direct passthrough：

- 不呼叫 ASR。
- 不呼叫 LLM/translation。
- 不寫 transcript。
- 不走 TTS synthesis 或 legacy passthrough queue。
- 只做必要 gain、limiter 與 boundary format conversion。

## ASR Language

Runtime 生效語言固定為：

```text
zh-TW, en, ja, ko, th
```

UI 不提供 `auto`。舊 config 的 `auto` 會 migration：

- `remote_asr_language: auto` -> `en`
- `local_asr_language: auto` -> `zh-TW`

ASR backend 主線維持 faster-whisper/CTranslate2。

## Audio Format

```yaml
audio:
  virtual_audio:
    sample_rate: 48000
    channels: 2
    bit_depth: 16
    dtype: int16
```

格式分層：

- app internal: float32 `AudioFrame`，保留 sample rate、channels、source metadata。
- ASR branch: `16000Hz mono float32`。
- Windows / bridge / driver boundary: `48000Hz / PCM16 / 2ch`。

不要把整個 app internal 改成 int16 stereo；int16 stereo 是外部 boundary contract。

## Runtime Assets

預設 runtime/model layout：

```text
runtimes/
  shared/
  faster_whisper/
  models/
    asr/
      large-v3-turbo/
    llm/
      hy-mt1.5-7b.gguf
  audio/
    sync_audio_bridge.exe
```

準備 runtime：

```powershell
powershell -ExecutionPolicy Bypass -File .\tools\runtime_setup\prepare_external_runtimes.ps1
```

檢查設定與 readiness：

```powershell
uv run python main.py --check
uv run python .\tools\validation\preflight_release_check.py --mode meeting
uv run python .\tools\validation\preflight_release_check.py --mode dialogue
```
