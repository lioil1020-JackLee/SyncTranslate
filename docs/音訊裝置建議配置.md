# 音訊裝置建議配置

本文件整理目前 SyncTranslate 搭配 LINE 視訊通話與 Voicemeeter 的建議接法。

目標是：

- LINE 麥克風只收到一種聲音
- 平常送翻譯後語音
- 若要直通，由 SyncTranslate 切換 `passthrough`
- 避免原始麥克風與翻譯語音同時混進 LINE 麥克風

## 1. 核心原則

- `B1` 只負責送給 LINE 麥克風
- `B2` 只負責把 LINE 對方聲音送給 SyncTranslate 當遠端輸入
- 本地麥克風直接給 SyncTranslate，不再另外進 Voicemeeter 的 `Stereo Input 1`
- SyncTranslate 決定送到 LINE 的內容是：
  - `tts`: 翻譯後語音
  - `passthrough`: 原音直通

## 2. LINE 設定

- 喇叭：`Voicemeeter AUX Input`
- 麥克風：`Voicemeeter Out B1`

這樣：

- LINE 對方聲音會先進 Voicemeeter 的 `AUX Input` strip
- LINE 自己收音只會吃 `B1`

## 3. SyncTranslate 設定

建議如下：

- 遠端輸入：`Voicemeeter Out B2`
- 遠端輸出：`Voicemeeter Input`
- 本地輸入：你的實體麥克風
- 本地輸出：你的實體耳機 / 喇叭

對應意義：

- `遠端輸入`
  - 讀取對方在 LINE 說話的聲音
- `遠端輸出`
  - 把要送回 LINE 的聲音送進 Voicemeeter
- `本地輸入`
  - 直接讀你的真實麥克風
- `本地輸出`
  - 播放翻給你聽的內容

## 4. Voicemeeter 設定

### 4.1 Stereo Input 1~3

目前建議：

- 先留空
- 尤其 `Stereo Input 1` 不要再選實體麥克風

原因：

- SyncTranslate 已直接抓取實體麥克風
- 若 Voicemeeter 再收一次同一支麥克風，容易讓路徑變複雜
- 也容易不小心把原始麥克風送進 `B1` 或 `B2`

### 4.2 Voicemeeter Input

這條是 SyncTranslate 的 `遠端輸出` 入口。

建議：

- 開 `B1`
- 關 `B2`
- `A1` 通常先關閉

作用：

- SyncTranslate 要送給 LINE 的聲音
  - 先進 `Voicemeeter Input`
  - 再由 `B1` 送給 LINE 麥克風

### 4.3 Voicemeeter AUX Input

這條是 LINE 喇叭聲音的入口。

建議：

- 開 `B2`
- 關 `B1`
- `A1` 可視需要開啟

作用：

- LINE 對方講話
  - 先進 `Voicemeeter AUX Input`
  - 再由 `B2` 給 SyncTranslate 當遠端輸入

### 4.4 Hardware Out

建議：

- `A1` 設為你的實體耳機 / 喇叭

## 5. 最終接線圖

### 遠端來話

`LINE 喇叭 -> Voicemeeter AUX Input -> B2 -> SyncTranslate 遠端輸入`

### 本地收音

`實體麥克風 -> SyncTranslate 本地輸入`

### 送回 LINE

`SyncTranslate 遠端輸出 -> Voicemeeter Input -> B1 -> LINE 麥克風`

## 6. 為什麼不要讓 LINE 直接吃實體麥克風

如果 LINE 直接吃實體麥克風，或 Voicemeeter 將實體麥克風也送進 `B1`，就容易出現：

- LINE 同時收到原始麥克風
- LINE 同時收到 SyncTranslate 輸出的翻譯聲音
- 原音與翻譯音混在一起

這不符合目前需求。

本配置的重點就是：

- LINE 麥克風只吃 `B1`
- `B1` 只承接 SyncTranslate 輸出的聲音

## 7. 什麼情況下才需要把麥克風接進 Voicemeeter

只有在你需要以下功能時，才考慮把實體麥克風掛進 `Stereo Input 1`：

- 想用 Voicemeeter 監聽自己的麥克風
- 想用 Voicemeeter 做 EQ / gate / comp
- 想讓其他軟體也從 Voicemeeter 拿麥克風
- 想保留一套人工備援路由

如果目前沒有這些需求，保持 `Stereo Input 1~3` 留空最乾淨。

## 8. 快速檢查清單

設定完成後，依序確認：

1. LINE 喇叭是否設成 `Voicemeeter AUX Input`
2. LINE 麥克風是否設成 `Voicemeeter Out B1`
3. SyncTranslate `遠端輸入` 是否設成 `Voicemeeter Out B2`
4. SyncTranslate `遠端輸出` 是否設成 `Voicemeeter Input`
5. SyncTranslate `本地輸入` 是否設成實體麥克風
6. `Voicemeeter AUX Input` strip 是否只開 `B2`
7. `Voicemeeter Input` strip 是否只開 `B1`
8. `Stereo Input 1~3` 是否沒有再掛同一支實體麥克風

## 9. 預期行為

當設定正確時：

- 對方講話時，SyncTranslate 的遠端字幕會更新
- 你講話時，SyncTranslate 的本地字幕會更新
- SyncTranslate 在 `tts` 模式下會把翻譯語音送回 LINE
- SyncTranslate 在 `passthrough` 模式下會把原音直通送回 LINE
- LINE 麥克風不會同時混入原始麥克風與翻譯語音
