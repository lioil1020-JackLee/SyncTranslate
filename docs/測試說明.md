# 測試說明

## 安裝

本地測試環境（推薦完整安裝）：

```powershell
# 1. 安裝主程式依賴
uv sync --locked

# 2. 準備外部 AI 運行時
powershell -ExecutionPolicy Bypass -File .\tools\runtime_setup\prepare_external_runtimes.ps1
```

如果只需要一般 Python 測試，可以跳過 `prepare_external_runtimes.ps1`，但某些 ASR 相關測試（faster_whisper_adapter、faster_whisper_adapter）會被 skip：

```powershell
uv sync --locked
```

## 執行方式

### 全部測試（pytest）

```powershell
uv run pytest -q
```

### 詳細輸出

```powershell
uv run pytest -v
```

### 單跑某幾組

```powershell
uv run pytest tests/test_asr_backend_routing.py tests/test_asr_streaming_and_profiles.py -v
```

### 系統檢查

```powershell
uv run python .\main.py --check
```

## 建議驗收流程

自動測試之外，這個專案仍建議搭配手動驗收：

1. 跑 `uv run pytest -q` 確認 all pass
2. 跑 `uv run python .\main.py --check`
3. 開啟程式確認 UI 能正常載入
4. 用中文單通道測試
5. 用雙通道同時發話測試
6. 匯出 `logs/session_reports/*.json`
7. 檢查：
   - `resolved_backend`
   - `device_effective`
   - `speech_started_count`
   - `soft_endpoint_count`
   - `hard_endpoint_count`
   - `endpoint_signal.pause_ms`
   - `partial_count`
   - `final_count`
   - `postprocessor.final.rejected_count`
   - `postprocessor.final.last_rejection_reason`
   - `dropped_chunks`

這些欄位通常比單看字幕內容更容易定位瓶頸。

## ASR benchmark 驗收

離線辨識目前已達穩定水準，日常改動優先看即時串流：

```powershell
# 英文即時串流回放
$audio = Get-ChildItem -Path downloads\benchmark\en -Filter *.wav | Select-Object -First 1
$ref = "downloads\benchmark_results\retest_20260416\english_reference_from_srt.txt"
uv run python tools\asr_benchmark\streaming_sim.py --config config.yaml --audio $audio.FullName --reference $ref --source remote --language en --speed 8 --output downloads\benchmark_results\streaming_en

# 中文即時串流回放
$audio = Get-ChildItem -Path downloads\benchmark\zh-TW -Filter *.wav | Select-Object -First 1
$ref = "downloads\benchmark_results\retest_20260416\chinese_reference_from_vtt.txt"
uv run python tools\asr_benchmark\streaming_sim.py --config config.yaml --audio $audio.FullName --reference $ref --source local --language zh-TW --speed 8 --output downloads\benchmark_results\streaming_zh
```

2026-04-25 實測參考：

- 英文即時：normalized accuracy 約 `95.7%`
- 中文長故事即時：約 `80%` 到 `83%`，會受背景音、片頭片尾與即時切段時機影響
- 完整測試：`uv run pytest -q` 為 `389 passed` 以上
