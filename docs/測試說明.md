# 測試說明

## 環境準備

先在專案根目錄同步 `uv` 依賴：

```powershell
uv sync --locked --extra local
```

如果只想跑不需要本地 Whisper 的測試，也可以先用：

```powershell
uv sync --locked
```

## 常用命令

完整測試：

```powershell
uv run pytest -q
```

以 `unittest` 方式逐案列出：

```powershell
uv run python -m unittest discover -s tests -p "test_*.py" -v
```

設定與裝置 smoke check：

```powershell
uv run python .\main.py --check
```
