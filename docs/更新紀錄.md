# 更新紀錄

## 2026-03-28

### UI 與互動

- 即時字幕頁新增明確的 `模式` 切換，將 `直通` 與 `翻譯` 從 `翻譯目標` 中拆開。
- `輸出音量` 改為獨立主音量，不再要求切模式時重新調整。
- 修正 `輸出音量` slider 的圓形 handle 與刻度未對齊問題。
- 恢復 `lioil.ico` 的執行期與打包 icon 路徑，避免視窗與 onedir 成品遺失圖示。
- 遠端翻譯列移除多餘的 `測試TTS` 按鈕，避免控件重疊。
- `模式` 已移到 `翻譯目標` 左側。
- 設定頁的快速操作區改成單行摘要，不再被截字。
- 修正 `模式` 切成 `直通` 後會被自動切回 `翻譯` 的問題。

### Runtime 與設定

- 預設參數回調為穩定優先。
- 不再強制使用錯誤的 `live_caption_stable` profile。
- 修正 `config.yaml` 中多組 LLM profile 被錯誤覆蓋的問題。
- `LiveCaptionPage.apply_config()` 現在會優先尊重顯式的 channel mode 設定，不再用目標語言與聲線值反推模式。

### 代碼清理

- 刪除 `LocalAiPage` 中舊版快速設定殘留方法。
- 刪除未使用的 `app/infra/audio/meter.py`。
- 清理 live caption 頁面中未使用的 helper 與 import。
- 將 `LiveCaptionPage` 重構為 channel-centric 寫法，收斂 remote/local 的重複 UI 與設定同步邏輯。
- 補齊進階設定中缺少的 runtime / LLM profile / TTS style 參數，並讓 `config.yaml` 能完整保存對應欄位。

### 文件

- 新增從零開始的 [安裝說明](./安裝說明.md)，包含 LM Studio、模型下載、SyncTranslate 啟動與第一次測試流程。

## 2026-03-26

### 基礎整理

- Runtime 固定採用 `bidirectional`。
- ASR / LLM 保持方向獨立。
- 移除 warmup / preheat 流程。
- 將 health check 統一命名為 system check。
